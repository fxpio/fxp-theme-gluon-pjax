Getting Started
===============

Prerequisites
-------------

This library uses:

- sonatra-theme-gluon
- all dependencies of sonatra-theme-gluon

Installation
------------

Installation is a quick:

Add library in your `bower.json`:

```json
{
    "dependencies": {
        "sonatra-theme-gluon-pjax": "latest"
    }
}
```

Launch bower:

```bash
$ bower install
```
